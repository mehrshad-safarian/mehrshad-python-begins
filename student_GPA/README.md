# GPA Calculator / محاسبه‌گر معدل

## English
GPA Calculator is a Windows application built with Python and Tkinter. 
It allows users to enter courses with their name, grade, and unit, 
view them in a table, and calculate the weighted GPA.

### Features
- Add courses with name, grade, and units
- Display all courses in a table
- Calculate weighted GPA
- Simple and user-friendly interface

### How to Run
1. Download the `GPA_Calculator.exe` from the `dist` folder.
2. Double-click the exe file to open the application.
3. Enter course name, grade, and unit, then click "Add Course".
4. Click "Calculate GPA" to see the result.

## فارسی
این برنامه یک اپلیکیشن ویندوزی است که با پایتون و Tkinter ساخته شده.  
کاربر می‌تواند درس‌ها را با نام، نمره و واحد وارد کند، آن‌ها را در جدول ببیند و معدل وزنی را محاسبه کند.

### ویژگی‌ها
- اضافه کردن درس با نام، نمره و واحد
- نمایش همه درس‌ها در جدول
- محاسبه معدل وزنی
- رابط کاربری ساده و کاربرپسند

### نحوه استفاده
1. فایل `GPA_Calculator.exe` را از پوشه `dist` دانلود کنید.
2. روی فایل exe دوبار کلیک کنید تا برنامه باز شود.
3. نام درس، نمره و واحد را وارد کنید و روی "Add Course" کلیک کنید.
4. روی "Calculate GPA" کلیک کنید تا نتیجه را ببینید.
